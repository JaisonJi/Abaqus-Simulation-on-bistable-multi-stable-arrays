# Abaqus-Simulation-on-bistable-multi-stable-arrays
INP files that can replicate the CAE simulations in Figure 4. 
To replicate the simulation, run the inp files with 'bulk' or 'B' at the end of file name. The buckling analysis introduce the geomatric imperfection into the structure. Then conduct the 'loading' inp to complete the simulation and obtain force-displacement responses.
